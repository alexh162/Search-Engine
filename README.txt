HOW TO USE:
1. Create an empty folder called "index_files" in the same directory as indexer.py
2. Run 'python3 duplicate.py' to create a set that contains near duplicate websites
3. Run 'python3 anchor_words' to traverse the json file to find anchor texts and the links they lead to
4. Run 'python3 indexer.py' to create the inverted indexes. There is an index for each letter in the alphabet and one for non-alphanumberic index
5. Run 'python3 searcher.py' to run a console search
6. Run 'python3 app.py' to run a web gui search
